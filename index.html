Sky Fight

Sky fight is a game I made in my 3rd year of university. It was built with processing, using Box2D and OSC. The aim is to survive for as long as you can as objects fall from above. You consist of a vulnerable hexagon, with an attached flail, which you can swing around. The chain does not hurt you but the business end does.

Later in the game, triangular enemies will join the fray. You can use your flail to hurt them, but likewise, so can they.

Boids Flocking Implementation

Boids is a relatively well known algorithm that models a seemingly complex, chaotic flock of constantly moving agents. This version was written in Java using Processing.

Each agent follows three simple rules: Alignment (try to go in the same direction as nearby agents), Cohesion (try to get close to nearby agents), and Separation (try to avoid getting too close to nearby agents).

I also made another version (featured at the top of the page), which I wrote in JavaScript.

Arduino Synthesizer

I had an interesting course in third year called Sensor-Based Interaction, an advanced programming class focusing on hardware, sensors, and interactive installations. The final project was to create something strange and interactive in a small team using three randomly chosen sensors.

We chose a synthesizer. A force sensor controlled the volume, a joystick determined the current cord, and a distance sensor determined the number of voices.
